# Desktop_App
README: Система управління замовленнями для кафе
Цей проєкт реалізує просту систему управління замовленнями для кафе з використанням консольного інтерфейсу на C#.

Основні функції
Додавання позицій: Додавання страв/напоїв з описом та ціною

Видалення позицій: Видалення пунктів замовлення за номером

Додавання чайових: Вибір між відсотковою ставкою або фіксованою сумою

Показати рахунок: Відображення детального рахунку з:

Сумою без податків

Сумою чайових

Податком GST (5%)

Загальною сумою

Очистити замовлення: Повне скидання поточного замовлення

Зберегти у файл: Експорт замовлення у текстовий файл

Завантажити з файлу: Імпорт замовлення з текстового файлу

Використання
Запустіть програму

Використовуйте меню для вибору операцій:

Введіть 1-8 для вибору пункту меню

Дотримуйтесь інструкцій на екрані

При роботі з файлами вкажіть повний шлях до файлу

Формат даних
Дані зберігаються у текстових файлах у форматі:

text
Опис|Ціна
Опис|Ціна
Tip|Сума_чаєвих
Приклад:

text
Cheeseburger|13.95
French Fries|4.50
Tip|3.29
